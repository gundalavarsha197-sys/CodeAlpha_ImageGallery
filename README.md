# 🌟 Advanced Image Gallery

This project is a **responsive and interactive Image Gallery Web Application** developed using **HTML, CSS, and JavaScript** as part of my Frontend Development Internship.

---

## 🚀 Features

✅ Responsive grid layout (mobile + desktop)
✅ Lightbox view for full image display
✅ Next / Previous navigation
✅ Keyboard controls (← → Esc)
✅ Image filtering by category
✅ Dark mode toggle 🌙
✅ Auto slideshow feature 🎞️
✅ Like/Favorite button ❤️
✅ Smooth animations and transitions

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)
